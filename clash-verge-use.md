# 准备工作
本软件采用python进行编写，依赖库有pystray和pillow

安装指令为：pip install pystray pillow

❗注意，如果软件启动不了的话，请安装依赖，由于我电脑环境复杂，未测试没安装依赖的话能不能打开软件。
